TrimSpreadsheet Demo

Demonstration of a Spreadsheet Engine written in accessible, unobtrusive JavaScript style. The formula calculation engine and user interface run completely in the browser. See more information at http://trimpath.com/project/wiki/TrimSpreadsheet.

Cells C7 and C8 have formulas. But, all cells are alive.
